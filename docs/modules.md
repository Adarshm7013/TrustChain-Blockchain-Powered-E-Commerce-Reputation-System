# System Modules

The TrustChain system is divided into multiple functional modules.

## 1. Identity Management Module
Handles buyer and seller registration, decentralized identity verification, and credential validation.

## 2. Product Management Module
Allows sellers to add, update, and manage products on the platform.

## 3. Purchase Management Module
Handles buyer purchases and transaction processing.

## 4. Feedback Token Module
Generates blockchain-backed feedback tokens after successful purchases.

## 5. Feedback Verification Module
Validates feedback tokens before allowing review submission.

## 6. Reputation Management Module
Calculates and updates seller reputation scores using smart contract logic.

## 7. Discount Token Module
Issues reward tokens after successful verified feedback submission.

## 8. Blockchain Ledger Module
Stores immutable transaction and reputation records.

## 9. Admin Monitoring Module
Allows monitoring of blockchain transactions, tokens, and reputation events.
